# Vulture

This library finds Unused And Dead Code In Python Projects.

### For installation
``
pip install vulture
``

### For using vulture
1. Go to Anaconda prompt or local cmd
2. Change your directory to the root folder
3. Run -->
```
vulture "filename".py
```
